================================
Symbol★Merged (데모)
- 양손에 든 물건(문양)을 합치고, 합친 문양의 능력을 활용하는, GBA용 퍼즐 플랫포머 게임.

Copyright 2021 Guyeon Yu (copyrat90@gmail.com)
All rights reserved.
================================

유의사항: 이 게임은 초기 데모 버전입니다.
미래에는 게임 세이브 파일 구조가 바뀔 수 있고, 그러면 기존 세이브는 쓸 수 없게 됩니다.

================================


이 게임은 GBA(게임 보이 어드밴스)에서 작동하는 게임입니다.

이 게임을 하시려면, [1. GBA 에뮬레이터] 혹은 [2. 진짜 GBA와 플래시 카트리지] 중 하나가 필요합니다.
이게 무슨 말인지 모르겠다면, 아래 링크를 눌러서 mGBA 에뮬레이터를 다운로드하시고, 실행한 후에, sym_merged.gba 파일을 mGBA 창에 끌어다 놓으세요.
https://mgba.io/


제 GitHub 레포지토리에 이 게임의 소스 코드를 공개해놨습니다. (아래 링크)
https://github.com/copyrat90/sym_merged
소스 코드는 MIT/Expat 라이선스 하에 배포됩니다.


이 게임은 Butano 엔진을 활용하여 만들어졌습니다.
Butano 엔진은 GBA 게임 개발용 Modern C++ 엔진입니다.
https://github.com/GValiente/butano
Butano 는 zlib 라이선스 하에 배포됩니다.

Butano 엔진이 없었다면, 여기까지 오진 못했을 겁니다.
GValiente님, 좋은 엔진을 만들어주셔서 감사합니다!


각종 애셋(그래픽, 음악, 효과음 등)의 라이선스는 애셋마다 다릅니다.
`licenses/asset` 폴더에서 각각의 라이선스를 확인해주세요.
(실제 애셋을 보고 싶다면, GitHub 레포지토리를 확인하시기 바랍니다.)

더 많은 문양과 더 많은 플랫포밍 기믹을 넣고 싶었지만,
첫번째 게임이라 그런지, 생각보다 시간이 훨씬 많이 걸려서,
기능을 다 넣을 수가 없었습니다.

아무튼, 이 게임을 다운로드해주셔서 감사하고, 즐거운 시간 되시길 바랍니다!
